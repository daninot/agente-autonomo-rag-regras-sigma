Generate a SIGMA rule to detect successful Cisco Duo MFA authentication
completed using a bypass code — temporary admin-issued passcodes that
attackers with admin access can abuse to defeat MFA without compromising the
enrolled device.

Target product: cisco; log source service: duo.

Detection matches Duo authentication log events where the authentication
reason indicates a bypass code was used.

References to include:
- https://duo.com/docs/adminapi#logs
- https://help.duo.com/s/article/6327?language=en_US
