Generate a SIGMA rule to detect suspicious Ruby on Rails exception classes in
application logs — indicators of exploitation attempts such as CSRF token
abuse, cross-origin forgery, invalid HTTP methods, or malformed requests.

Target product: ruby_on_rails; log source category: application.

Detection uses keyword matching against ActionController exception class names.
