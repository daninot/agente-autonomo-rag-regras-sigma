Generate a SIGMA rule to detect local file read vulnerability exploitation in
JVM-based applications — triggered when user input causes file-not-found
exceptions containing path traversal sequences in error logs.

Target product: jvm; log source category: application. Requires error logs at
LOG_LEVEL ERROR or above — document this in the rule.

Logs are unstructured. Detection must use a keyword block with the '|all'
modifier, requiring a Java file exception class AND a path traversal pattern
to co-occur in the same entry.
