Generate a SIGMA rule to detect registry reconnaissance via reg.exe —
adversaries query sensitive registry keys to gather credentials, persistence
mechanisms, and service configurations.

Target product: windows; log source category: process_creation.

Detection requires THREE selections with AND (all of selection_*): (a) process
is reg.exe matching both Image path and OriginalFileName to cover renamed
binaries; (b) command line contains the query subcommand; (c) command line
targets sensitive registry paths covering run keys, services, shell loading,
policies, and logon configuration.
