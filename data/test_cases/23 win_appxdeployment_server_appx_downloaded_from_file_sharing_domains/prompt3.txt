Generate a SIGMA rule to detect an AppX package queued for installation that
was downloaded from a file sharing or CDN domain — abused by malware campaigns
like BazarLoader to deliver malicious Windows apps via trusted mechanisms.

Target product: windows; log source service: appxdeployment-server.

Detection matches a specific AppX deployment EventID combined with a path
containing known file sharing, paste, or CDN domains (GitHub, Discord CDN,
Dropbox, Mega, Pastebin, and similar platforms commonly abused for malware
staging) using |contains.
