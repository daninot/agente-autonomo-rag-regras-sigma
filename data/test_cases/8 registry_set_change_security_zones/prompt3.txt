Generate a SIGMA rule to detect registry modifications that add a domain to
the Internet Explorer ZoneMap trusted sites — lowering security controls for
attacker-controlled domains and enabling defense evasion or persistence via
browser trust abuse.

Target product: windows; log source category: registry_set.

Detection matches registry write events targeting the IE ZoneMap Domains
path, but must EXCLUDE legitimate zone assignments (My Computer, Local
Intranet, and empty values) — use exclusion logic in the condition.
