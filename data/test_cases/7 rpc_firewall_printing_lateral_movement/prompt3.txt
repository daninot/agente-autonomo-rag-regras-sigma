Generate a SIGMA rule to detect remote RPC calls abusing the Windows remote
printing services (MS-RPRN / MS-PAR) for lateral movement — the attack class
popularized by PrintNightmare (CVE-2021-34527).

Target product: rpc_firewall; log source category: application. This rule
targets the RPC Firewall product (Zero Networks), which monitors RPC interface
invocations. It is NOT a rule against native Windows Print Spooler logs.

The RPC Firewall must be configured to audit the MS-RPRN and MS-PAR interface
UUIDs. Detection matches the structured event emitted by the RPC Firewall when
a monitored RPC interface is invoked, filtering by the specific interface UUIDs
that correspond to the printing protocols.
