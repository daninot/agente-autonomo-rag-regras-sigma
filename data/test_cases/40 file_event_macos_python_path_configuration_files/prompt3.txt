Generate a SIGMA rule to detect creation of Python path configuration files
(.pth) in macOS Python library folders — these execute at every Python startup
(3.5+) regardless of import, enabling code execution and persistence. Observed
in CVE-2024-3400 (Palo Alto GlobalProtect) exploitation.

Target product: macos; log source category: file_event.

Detection combines: (a) regex match (|re) on the target filename for the
Python site-packages directory across versions 3.5+, case-insensitive,
covering both single- and double-digit minor versions; (b) |endswith on the
.pth extension.
