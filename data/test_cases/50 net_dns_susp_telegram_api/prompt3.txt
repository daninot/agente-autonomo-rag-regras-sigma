Generate a SIGMA rule to detect DNS queries to the Telegram Bot API endpoint —
malware families (TeleRAT, TeleCrypt, KillDisk) abuse Telegram bots as a
covert C2 channel that blends with legitimate messaging traffic.

Log source category: dns.

Detection matches DNS queries for the Telegram Bot API hostname using
exact-match (no modifier) on the query field.
