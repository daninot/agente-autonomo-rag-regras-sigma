Generate a SIGMA rule to detect delete actions in GitHub audit logs targeting
codespaces, environments, projects, and repositories — indicating potential
destructive activity or unauthorized resource removal.

Target product: github; log source service: audit. Requires GitHub audit log
streaming to be enabled — document this in the rule, including the setup
documentation URL.
